# Hospitale Dashboard 

## Overview

This project is a Hospital Management System that allows for the management and analysis of Hospital data. The system provides various features including viewing Hospital data, searching, and filtering based on specific criteria.

## Version 0.0.3
## Features
- Date Range Filtering: Select a custom date range on the page to filter patient data accordingly.
- Enhanced search functionality within the table for more precise results.
- New filter options to filter patient data by gender and date range.
- Updated visual design of the table for a better user experience.
- Added chart view to display patient data by gender and track creation trends by month.
