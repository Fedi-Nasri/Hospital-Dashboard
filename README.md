# Hospitale Dashboard 

## Overview

This project is a Hospital Management System that allows for the management and analysis of Hospital data. The system provides various features including viewing Hospital data, searching, and filtering based on specific criteria.

## Version 0.0.2

### Features
- **Table Integration**: Connected to the database to retrieve all patient data.
- **Chart Functionality**: Implemented a chart feature to display patients based on a specific date range.
- **Chart Types**:
  - **Patient Chart**: Displays all patients.
  - **Maladichart**: Displays Diseases with not connecting to any controller or db.
- **Search Functionality**: Added a search option within the table to filter data.


## Working on  Version 0.0.3 Comming Soon  New Features and Enhancements

- **Date Range Filtering**: Added functionality to select a date range on the page and filter data based on this range.
- **Table Enhancements**:
  - Improved search options within the table.
  - Added new filter options:
    - **Gender**: Filter patients based on gender.
    - **Date Range**: Refine search results by date range.
    - **CNSS Status**: Filter patients by CNSS (social security number) presence.
- **Styling Improvements**: Enhanced the visual styling of the table for better user experience.
